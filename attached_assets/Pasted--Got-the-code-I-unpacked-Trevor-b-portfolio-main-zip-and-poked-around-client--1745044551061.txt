1. Branding layer

‑ tailwind.config.ts

extend: {
+ colors: {
+   brandBg:  "#150734",   // deep space violet
+   brandHi:  "#D2FF4C",   // aurora lime
+   brandFx:  "#3CFBFF",   // electric cyan
+ },
  ...
}

Dark mode stays (dark: prefix) but now you’ve got three brand tokens to reuse in every component.

Add to globals.css:

:root {
  --c-bg: theme(colors.brandBg);
  --c-hi: theme(colors.brandHi);
  --c-fx: theme(colors.brandFx);
}
